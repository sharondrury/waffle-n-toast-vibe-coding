Feature: Feat-005 | Carousel — auto-scroll animation
------------------------------------------------------
What was implemented:
Files changed:
  vibe-coding/src/website/pages/homePage/showcase/showcase.scss
  vibe-coding/src/website/pages/homePage/showcase/showcase.jsx
Engine components considered:
  - timer.jsx — reviewed; not used. CSS @keyframes marquee handles the loop
    natively without a JS timer, which avoids frame-rate jank and is simpler
    to pause/resume via animation-play-state.
Note: @keyframes marquee translates the track from translateX(0) to
translateX(-50%). Because the array is doubled, -50% lands exactly at the
start of the second copy, making the reset invisible. Duration set to 35s.

Feature: Feat-006 | Carousel — pause, hover and drag interactions
------------------------------------------------------------------
What was implemented:
Files changed:
  vibe-coding/src/website/pages/homePage/showcase/showcase.jsx
  vibe-coding/src/website/pages/homePage/showcase/showcase.scss
Engine components considered:
  -
Note: isPaused state drives animation-play-state: paused / running.
Drag is handled via onMouseDown / onMouseMove / onMouseUp on the track ref.
A isDragging ref (not state) avoids unnecessary re-renders during drag.
currentTranslate ref stores the drag offset and is applied as an inline
transform during drag; on mouse-up the CSS animation resumes from the
visual position.
