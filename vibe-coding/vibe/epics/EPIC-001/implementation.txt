Learning Notes
---------------
  - NavLink applies an `active` class automatically when the URL matches the `to`
    prop — no manual state tracking needed for active link styling
  - CSS animation via @keyframes conflicts with inline style.transform. For
    carousels that need drag (EPIC-005), switch to a RAF loop instead
  - position: absolute + left: 50% + translateX(-50%) is the cleanest way to
    true-centre a flex child when siblings have unequal widths
  - Sticky footer: flex column on body/root/app + flex: 1 on main is the most
    reliable approach across all page lengths
