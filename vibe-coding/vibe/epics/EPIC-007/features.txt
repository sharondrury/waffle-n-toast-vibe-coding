Feat-004 | contactValidation.js
---------------------------------
File: vibe-coding/src/website/pages/homePage/contactSection/contactValidation.js

Pure JS module — no React, no JSX. Exports a single validate function.

Rules:
  - name: required
  - companyName: optional
  - email: required + valid format
  - message: required
  - Returns an errors object — empty means valid, keys match field names

  Status: done

Feat-005 | contactForm.jsx
----------------------------
File: vibe-coding/src/website/pages/homePage/contactSection/contactForm.jsx

Self-contained form component used in both the inline contact section
and the Get In Touch modal. Props:
  - onSuccess (function) — called after successful EmailJS send
  - onError   (function) — called after failed send (optional, can show inline)

Internal state:
  fields, errors, sending, submitStatus ('idle' | 'success' | 'error')

Fields rendered:
  1. Name           — text input, required
  2. Company Name   — text input, optional
  3. Email          — email input, required
  4. Message        — textarea, required

EmailJS send:
  Service: service_kg75p6l
  Template: template_hkes3vq
  Params: from_name, company_name, reply_to, message
  Key: import.meta.env.VITE_EMAILJS_PUBLIC_KEY

  Status: done
